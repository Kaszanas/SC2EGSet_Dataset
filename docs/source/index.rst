Welcome to sc2egset_dataset's documentation!
============================================

This documentation describes the official API for interacting with SC2EGSet: StarCraft II Esport Game State Dataset. Our dataset was built on top of the SC2ReSet: StarCraft II Replaypack Set. We provide PyTorch and PyTorch Lightning abstractions for the dataset download, unpacking, loading, and manipulation.

.. warning:: Our project is actively developed. If you find bugs, please report them here: https://github.com/Kaszanas/SC2EGSet_Dataset

Installation
--------

.. note:: We are working on a version that will be distributed using PyPi.

Currently the installation requires cloning our repository and manual requirements installation. Please follow the steps below.

1. Clone the repository:

.. code-block::

   git clone https://github.com/Kaszanas/SC2EGSet_Dataset.git

2. Do something more
